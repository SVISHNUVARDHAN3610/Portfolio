Fine-Tuning T5 model pure pytorch

Integrated Low-Rank Adaptation (LoRA) into the T5-small model for efficient parameter-efficient fine-tuning on the CoNLL-2003 Named Entity Recognition dataset using pure PyTorch, without external fine-tuning libraries. Designed and implemented custom LoRA modules for T5 attention layers, reducing trainable parameters by over 90% while maintaining comparable F1-score to full fine-tuning approache. Constructed a modular training pipeline with logging, masking, gradient clipping, and checkpointing, enabling fast experimentation and reusable components for future low-resource NLP tasks.
